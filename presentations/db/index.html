DDL - Data Definition Language

  • CREATE

    Używany do tworzenia struktur w bazie

    przykład użycia:

    CREATE TABLE customers (ID int NOT NULL, first_name VARCHAR(50) NOT NULL, last_name VARCHAR(100) NOT NULL, PRIMARY KEY (ID))

DDL - Data Definition Language

  • DROP

    Używany do usuwania struktur z bazy

    przykład użycia:

    DROP TABLE customers

DDL - Data Definition Language

  • ALTER

    Używany do zmiany struktur w bazie

    przykład użycia:

    ALTER TABLE customers ADD COLUMN city

DQL - Data Query Language

  • SELECT

    Używany do wyciągania danych z bazy

    przykład użycia:

    SELECT {CO} FROM {SKĄD}

    SELECT first_name, last_name FROM customers

DML - Data Manipulation Language

  • INSERT

    Używany do wpisywania nowych danych do bazy

    przykład użycia:

    INSERT INTO {DOKĄD} (KOLUMNA_1, KOLUMNA_2) VALUES (WARTOŚĆ_1, WARTOŚĆ_2)

    INSERT INTO customers (first_name, last_name) VALUES ('Jan', 'Kowalski')

DML - Data Manipulation Language

  • UPDATE

    Używany do aktualizowania istniejących danych w bazie

    przykład użycia:

    UPDATE {CO} SET {KOLUMNA_1} = {WARTOŚĆ_1}

    UPDATE customers SET last_name = 'Nowak' WHERE last_name = 'Kowalski'

DML - Data Manipulation Language

  • DELETE

    Używany do usuwania danych z bazy

    przykład użycia:

    DELETE FROM {SKĄD}

    DELETE FROM customers WHERE last_name = 'Kowalski'

DCL - Data Control Language

  • GRANT

    Używany do nadawania uprawnień do obiektów użytkownikom

    przykład użycia:

    GRANT ALL PRIVILEGES ON customers TO developer WITH GRANT OPTION

DCL - Data Control Language

  • REVOKE

    Używany do odbierania uprawnień do obiektów użytkownikom

    przykład użycia:

    REVOKE ALL PRIVILEGES ON customers FROM developer

DCL - Data Control Language

  • DENY

    Używany do zabraniania wykonywania operacji

    przykład użycia:

    DENY SELECT ON customers TO developer
